## 如何使用yclust
通过subcommand子命令实现不同功能
### subA: sketch
usage: ./yclust sketch <-i> input_FA_name <-o> output_sketch_filename [-Options]
guide: see ./yclust sketch -h for more informations

### subB: cluster
usage: ./yclust cluster <-i> input_FA_name <-S> input_sketch_filename <-o> output_cluster_filename [-Options]
guide: see ./yclust cluster -h for more informations
